# Sonic Canvas

This is a Next.js portfolio website for an aspiring sound engineer, "Sonic Canvas".

## Core Features

- **Project Showcase**: Displays music projects with detailed descriptions, audio embeds, and screenshots.
- **Keyboard Performance Videos**: Embeds videos of keyboard performances.
- **Certification Verification**: Showcases Trinity College London certifications.
- **Interactive Bio**: A dynamically updated biography section generated with Genkit AI.
- **Contact Form**: An easy-to-use form for inquiries.
- **Skills & Experience**: Highlights technical skills and professional experience in audio production and live performance.

To get started, run the development server:

```bash
npm run dev
```

The main page is located at `src/app/page.tsx`.
